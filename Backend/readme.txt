Comandos para adicionar ferramentas de backend
--------------------------------------------------------------------------------
1.1 > dart pub add functions_framework

--------------------------------------------------------------------------------
2.1 > dart pub global activate dartfn
2.2 -> Adicionar o path mostrado após a execução do comando colocar na 
          variável de ambiente do meu computador.
2.3 > dartfn generate -h
2.4 > dartfn generate helloWorld

--------------------------------------------------------------------------------
3.1 # Criar um serviço http
3.2 > dart pub add shelf

--------------------------------------------------------------------------------
4.1 # Criar um serviço http usando rotas na função baseado no nodeJs
4.2 > dart pub add shelf_router

--------------------------------------------------------------------------------
5.1 # Criar classes com funções baseado no JAVA - API
5.2 > dart pub add shelf_router_generator

6.1 # Compilar pra liguangem de maquina
6.2 > dart compile exe bin/server.dart -o bin/server.exe
